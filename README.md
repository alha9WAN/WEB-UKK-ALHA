# Sistem Peminjaman Alat

Aplikasi berbasis Laravel untuk mengelola peminjaman alat secara online.

## Fitur Utama
- Register & Login User
- Verifikasi Email
- Booking / Peminjaman Alat
- Pembayaran Online (Midtrans Sandbox)
- Notifikasi Transaksi
- Dashboard User & Petugas & Admin
- Persetujuan Dan Penolakan Peminjaman
- Pengembalian Alat

## Alur Sistem
1. User register & verifikasi email
2. Login ke sistem
3. Pilih alat & lakukan peminjaman
4. Melakukan pembayaran via Midtrans
5. Menunggu persetujuan petugas
6. Menggunakan alat
7. Melakukan pengembalian

## Teknologi
- Laravel
- MySQL
- Midtrans API
